centos搭建pyspark环境

lucky 2024-06-09 14:53:21
Categories： Tags：

注意：每次配置变量完记得source生效

1.操作系统安装**

（1）操作系统镜像

下载镜像 CentOS-7-x86_64-DVD-1810-centos7.6.iso

（2）BMC登录

1.使用浏览器访问：192.168.2.100 。用户名：Administrator 密码：Admin@9000

3、登录IBMC后可以查看服务器详细信息

4、点击远程控制，选择JAVA集成远程控制台(虚拟KVM)，使用JAVA程序显示服务器界面

（3）操作系统安装

1.打开vmware,选择创建虚拟机

\2. 选择自定义，点击下一步

\3. 选择稍后安装操作系统，点击下一步

\4. 点击下一步，选择linux，版本选择Centos7

\5. 点击下一步，修改虚拟机名称slave**/master,选择安装位置，安装在G:\VM下面，点击下一步，选择继续

\6. 处理器数量选择2 ，核数选择2，点击下一步

\7. 点击下一步，内存选择2G

\8. 点击下一步，选择使用网络地址转换NAT

\9. 选择下一步，选择默认，点击下一步

\10. 选择创建新虚拟磁盘

\11. 点击下一步

\12. 点击完成

\13. 打开创建的虚拟机，添加iso

\14. 开启虚拟机

\15. 选择安装系统

\16. 选择continue

\17. 选择data

\18. 选择data,修改为上海时间

\19. 选择done,选择software selection

\20. 选择done

\21. 选择done,选择accept

\22. 开始安装

设置密码hadoop（用户root）

2.Linux主机基础配置

（1）网络配置 nmtui

1.查看网卡

2.配置网卡ens33

​ 检查对应的网卡速度：ethtool eno4(网卡名称)

​ GE1的用作网络管理，25G、100GE用做业务网络，分别修改他们的配置文件

​ vi /etc/sysconfig/network-scripts/ifcfg-ens33

​ vi /etc/sysconfig/network-scripts/ifcfg-enp134s0f1

（4）关闭防火墙

​ iptables -F

​ systemctl stop firewalld.service

​ systemctl disable firewalld.service

3.安装jdk1.8

​ 安装包:

​ jdk-8u292-linux-x64.tar.gz

（1）将软件包放入/opt下进行解压

tar -zxvf jdk-8u212-linux-x64.tar.gz

（2）将解压的软件包放入/usr/local下

mv jdk1.8.0_292/ java

mv java /usr/local/java

（3）修改环境变量

vi /etc/profile(最底部添加)

export JAVA_HOME=/usr/local/java

export JRE_HOME=$JAVA_HOME/jre

export PATH=$JAVA_HOME/bin:$PATH

export CLASSPATH=$CLASSPATH:$JAVA_HOME/lib:$JAVA_HOME/jre/lib

使环境变量生效

source /etc/profile

验证java是否安装成功

4. Python安装

上传！解压！

tar -zxvf Python-3.7.7.tgz

将解压的文件放置在/usr/local/python3/bin/python3路径下

mv Python-3.7.7 /usr/local/python3/bin/python3

创建软链接

ln -s /usr/local/python3/bin/python3 /usr/local/bin/python3

ln -s /usr/local/python3/bin/pip3 /usr/local/bin/pip3

1、Spark安装

上传，解压

环境变量即可

vi /etc/profile

export SPARK_PYTHON=/usr/local/bin/python3

export SPARK_HOME=/root/spark/pakeage/spark-3.2.0-bin-hadoop3.2

source /etc/profile

验证

进入/opt/spark-2.4.5-bin-hadoop2.7/bin目录

./pyspark
