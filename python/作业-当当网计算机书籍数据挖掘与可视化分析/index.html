一、摘要

本研究通过scrapy爬取当当网中关于计算机的书籍数据进行数据挖掘与可视化分析，系统地分析了当当网计算机书籍的销售数据。我们从多个维度（包括出版社、出版年份、打折力度等）对书籍数据进行了详细的探索，旨在揭示市场趋势和读者偏好。研究结果显示，清华大学出版社和人民邮电出版社在市场中占据主导地位，计算机书籍的出版数量在2005年至2011年间经历了显著增长，而近几年则呈现出波动和下降的趋势。通过分析书籍价格和打折力度的分布情况，我们发现大部分书籍的价格集中在低价区间，且打折力度适中。这些发现为出版社和书店在未来的出版和营销策略提供了数据支持。

二、研究背景与意义

随着信息技术的飞速发展，计算机书籍市场也随之蓬勃发展。互联网的普及使得在线书店成为人们购买书籍的主要渠道之一。当当网作为国内领先的在线书店，其销售数据能够反映出市场的动态变化和读者的需求偏好。通过对这些数据进行挖掘与分析，不仅可以帮助出版社优化出版策略，还能为书店的营销和库存管理提供科学依据。

本研究的主要意义在于：

  1. 市场分析：通过对当当网计算机书籍销售数据的分析，揭示了市场的整体趋势和变化，为出版社了解市场需求提供了重要参考。
  2. 决策支持：为出版社和书店在书籍出版和销售策略的制定提供数据支持，帮助其在激烈的市场竞争中做出科学决策。
  3. 读者行为研究：通过对读者购买行为的分析，了解读者的需求和偏好，帮助书店和出版社更好地服务读者。

六、建立数据挖掘模型（K-means）

我们使用了k-means聚类算法，循环计算不同聚类数下的 Silhouette Score，选出最优的n_clusters值，训练模型得出数据的n_clusters值增加cluster列，根据cluster的不同进行分类，将不同类别的title使用jieba分词选出词频率最高的作为cluster的类别，最后进行可视化分析。

1.特征选择

由于价格分布图可知price/pre_price几乎都在0.7-0.8之间，pre_price作为特征值的意义不大，critic中不同的店铺售卖的书籍大部分相同，author统计时发现出书最多的作者只有7本没有参考性，public跟书籍分类没有关系，每年都有不同类型的书籍出版time也没有参考意义，综上我们只选择了 [‘price’, ‘title’]作为特征。

但由于title为字符串，我们对title去重然后使用hash将书名转化为数值，再使用归一化将数值缩小以便于训练，两次处理后的title新增一列为title_hash_normalized作为训练特征

2.n_clusters选择

我们循环输出轮廓系数得出n_clusters=4是轮廓系数最好

5.聚类结果

1.簇的分布和特征分析

教程（红色）：大多数书籍的价格在100元以下，标题哈希值集中在-1到1之间。这表明教程类书籍通常价格较低且标题多样化程度较大，有计算机相关各种各类的教程。

应用（绿色）：书籍的价格主要在100元以下，标题哈希值大部分集中在-1到0之间，说明应用类的书籍的价格相对较低，即应用类书籍的标题较为集中，围绕某个领域展开因此标题多样性相对较小。

视觉（蓝色）：这类书籍的价格范围较大，从低到高都有覆盖，标题哈希值在-1到2之间，这计算机视觉cv类书籍在价格和标题多样性上都比较广泛。

研究（青色）：价格主要集中在300元以上，标题哈希值集中在-1到2之间，显示出这些书籍的高价特性和多样化标题，同时表明研究类型书籍在市场中的成本和价值较高，少量分布点也可以看出研究类书籍及其少量。

2.价格和类别的关系

教程和应用类书籍价格较低，适合入门级或广泛读者群体。

视觉类书籍价格跨度较大，可能包括从基础入门到专业深度的各种类型。

研究类书籍价格较高，可能是专业性较强或高级研究类的书籍。

3.Silhouette Score

轮廓系数 为0.5159，表明聚类效果较好。Silhouette Score在0到1之间，接近1表示聚类效果好，各簇之间分离度高，簇内紧密度高。

1.书籍关键词词云图

拆分书名，去除空格等无效字符，过滤单个字符和没有意义的数字，生成新数据作为数据源生成词云图

关键词频率

  1. 高频词：

    词云图中的字体大小表示词语的频率，字体越大表示词语出现的频率越高。

    例如，”基础”、”计算机”、”应用”、”教程”、”网络” 这些词语非常大，说明它们在文本中出现的频率最高。

  2. 中频词：

    一些词语的字体大小适中，表示它们在文本中出现的频率中等。

    例如，”实验”、”指导”、”实践”、”计算机科学”、”项目”、”软件”、”硬件”等。

  3. 低频词：

    词云图中较小的词语表示它们在文本中出现的频率较低。

    例如，”案例”、”系统”、”分析”、”维护”、”操作系统”等。

说明在计算机相关书籍中，计算基础教程类书籍占比较大，案例分析等应用书籍占比较小，可能与市场计算机入门者多需求量大等因素相关。

稳定期：2012年至2015年，出版数量相对稳定。

波动期：2016年至2023年，出版数量波动较大，但总体上仍保持在较高水平。

近期的显著下降：2024年，出版数量显著下降，可能受多种因素影响，如市场需求变化、出版周期等。

这种变化趋势反映了计算机书籍市场的动态变化，可能与技术发展、市场需求、出版策略等多种因素有关。

    价格峰值的出现可能与特定书籍的出版或者市场需求的变化有关。

  2. 价格波动性：

    从图中可以看到，近些年（尤其是2015年之后），价格波动显著增加。

    波动性增加意味着市场上书籍的定价更加多样化，可能受到不同书籍类别、市场需求以及其他因素的影响。

    总体来看，从2000年到2024年，书籍的平均价格呈现逐步上涨的趋势。
