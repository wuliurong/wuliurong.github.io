github基础语句

lucky 2023-06-05 13:01:15
Categories： Tags：

查看仓库状态
git status

初始化仓库
git init

（红色文件）添加文件到Git仓库（状态变红色），文件添加到临时缓冲区
git add test.txt

将缓冲区文件提交到Git仓库
git commit -m “文件备注”

打印Git仓库日志
git log

查看仓库分支
git branch

创建分支
git checkout newbranchname

切换分支
git checkout branchname

分支合并,将a分支合并到当前分支
git merge a

删除分支
git branch -d delbranchname

给当前的分支添加标签，可以通过标签名切换分支
git tag tagname

本地Git与Github绑定
生成SSH key

查看是否安装ssh
ssh

生成密钥，三次回车，生成两个文件
id_rsa 和 id_rsa.pub 隐藏文件

将id_rsa.pub内容添加到Github setting SSHkey中

查看是否绑定成功
ssh -T git@github.com

本地仓库链接到github上
git remote add origin http://仓库url地址

本地上传到github
git push origin master

github同步到本地
git pull origin master

克隆远程仓库地址
git clone https//:

第一次向远程仓库提交代码需要输入账号和密码
